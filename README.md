### 시작하기
---

1. 파일 복제(npm install 필수)
```
$ git clone --branch dev01 https://github.com/anonypatron/skuniv-contest-front.git
$ cd skuniv-contest-front
$ npm install
```

2. 환경변수 설정하기


   - 루트 디렉터리(skuniv-contest) 아래에 .env파일만들기
   - .env파일에 아래내용 복사하기

``` 
VITE_API_URL=http://localhost:8080
```

3. 실행(localhost:5173)

```
$ npm run dev
```
